About the system

My name is Prashant Mishra, and I am a Computer Science Engineering student at Lovely Professional University (LPU), with a minor specialization in Data Science.

Secure File Management System is a web-based application designed to provide a safe, efficient, and user-friendly platform for storing and managing files. The system integrates military-grade AES-256-CBC encryption, JWT-based authentication, TOTP two-factor authentication, real-time malware scanning, and buffer overflow detection.

Users can securely upload, download, share, and manage files with full access control. The security log panel provides transparency into all detected threats.
